Cryptomator

Criado em12-09-2022|Atualizado em20-10-2022|Informática
|Visualizações:

O Cryptomator é uma ferramenta leve de criptografia para a proteção de dados em nuvem. Utiliza um método avançado de criptografia (AES) e também protege contra ataques de força bruta. É código aberto, para que qualquer pessoa possa baixar e revisar o código fonte.

A criptografia de cada arquivo é feita individualmente, de modo que a edição de apenas um pequeno arquivo de texto alterará o arquivo criptografado correspondente.

Se você está procurando uma maneira fácil e confiável de proteger seus dados na nuvem, o Cryptomator pode ser a escolha certa.

Como funciona o Cryptomator?

O Cryptomator armazena dados sensíveis em um drive virtual chamado Vault. Esta unidade pode ser usada como uma unidade flash USB normal ou disco rígido. Tudo que você coloca no cofre é criptografado e protegido por uma senha. Estes cofres podem ser armazenados em um disco rígido local, na nuvem, um disco rígido externo ou pen drive.

Uma vez que os cofres são colocados na nuvem, como Google Drive ou Dropbox, os dados podem ser acessados de todos os dispositivos com conexão à internet usando o aplicativo de desktop Cryptomator ou o app para celulares.

Note que o Cryptomator não se conecta diretamente aos serviços de armazenamento em nuvem: Se você quiser criptografar seu armazenamento em nuvem usando o Cryptomator, você precisará instalar um cliente de sincronização (por exemplo, Dropbox, Mega, Google Drive) ou montar seu próprio servidor com compartilhamento de rede.

Instalando o Cryptomator.

Cryptomator funciona em quase todos os sistemas operacionais modernos, incluindo GNU/Linux, MS Windows e Mac OS X. O aplicativo móvel do Cryptomator também está disponível.

Arch Linux e seus derivados, como Antergos e Manjaro Linux, estão disponíveis nos seguintes URLs. https://aur.archlinux.org/packages/cryptomator/ Pode ser instalado utilizando as ferramentas AUR helper

_$ yay -S cryptomator_

Para Ubuntu Linux Mint, Elementary OS e outros derivados podem ser instalados usando o seguinte PPA:

_$ sudo add-apt-repository ppa:sebastian-stenzel/cryptomator_

_$ sudo apt-get update_

_$ sudo apt-get install cryptomator_

Para outras distribuições Linux, baixe e instale o arquivo de configuração na página de download.

Usando o criptomador

Uma vez instalado, inicie o Cryptomator a partir do menu. É normalmente encontrada em Menu->Acessórios->Criptomador.

Esta é a aparência do Cryptomator

Como você pode ver na imagem acima, já tenho alguns cofres no momento. Vamos criar um novo.

Para criar um novo cofre, aperte no canto inferior esquerdo + e selecione “Criar novo cofre”. Em seguida, selecione um local, digite um nome para seu novo cofre e clique em Salvar.

Proteja-o com uma senha forte. Certifique-se de que a senha contenha pelo menos 10 caracteres.

Acessando o cofre

Selecione e abra o cofre, digite a senha e clique em Unlock Vault. Clique em Mais opções abaixo do campo de senha para opções adicionais como salvar a senha (você não precisa digitar a senha toda vez que acessar o cofre), desbloquear automaticamente o cofre na inicialização, montar/desmontar o cofre e tornar o acionamento público.

O cofre é montado como um drive virtual e aparece no Gerenciador de Arquivos. Esta unidade pode ser usada como uma forma de armazenar dados sensíveis usando uma unidade local.

Se o cofre já estiver desbloqueado, você pode encontrá-lo no Explorador de Arquivos. Também pode ser acessado usando o Cryptomator. Para fazer isso, selecione o cofre desbloqueado da lista, clique na pequena seta para baixo ao lado de ‘Lock vault’ e selecione ‘View drive’. Você também pode copiar a URL WebDAV da lista suspensa e usá-la no cliente WebDAV selecionado para acessar o cofre.

Fechando o cofre

Quando você tiver terminado de usar o cofre, você pode travar o Cofre com segurança clicando na opção Lock Vault.

Como você pode ver, o Cryptomator é muito fácil de usar e configurar. Se você está procurando uma ferramenta de criptografia em nuvem simples, fácil de usar e totalmente funcional, recomendo utilizar o Cryptomator.
