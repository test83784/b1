Fingerprint

Criado em11-10-2022|Atualizado em21-10-2022|Informática
|Visualizações:

Enquanto você navega em um website, você já viu anúncios baseados em websites que você já visitou antes? Isso acontece porque as empresas de publicidade acompanham o seu comportamento na web.
Um dos métodos mais conhecidos de rastreamento é através de cookies HTTP, mas este artigo se concentra no uso do fingerprint (“impressão digital”) que na web podem significar ‘informações que identificam o dispositivo do usuário de um site’.
O Web Browser Fingerprint está presente em sites de todo o mundo, inclusive na NBC News nos EUA e na BBC no Reino Unido. Também estão em redes sociais como o Facebook, Instagram, Twitter, Youtube, etc. Mesmo alterando o IP através de um VPN e limpando o cache do navegador, esses sites ainda conseguem identificar os usuários.
Os administradores do site podem obter informações como o tipo de navegador, resolução de tela, nomes de plugins e lista de fontes instaladas dos usuários que acessaram o site. Todo esse conjunto de características formam o fingerprint. Estas informações são ligeiramente diferentes para cada usuário e, portanto, são pontos característicos que podem ser usados na identificação do visitante.
Se apenas uma informação for detectada, é difícil identificar os usuários porque há várias pessoas usando o mesmo navegador e a mesma resolução de tela, mas se várias caracteíriscas forem combinadas, é raro que todos os pontos de característica coincidam, de modo que podem ser usados para identificar os usuários.
Técnicas de evitar o fingerprint

FireGloves
FireGloves é uma extensão do Firefox que permite que o valor da fingerprint do usuário seja alterado para um valor diferente cada vez que ele visita um website. Por outro lado, essa mudança pode fazer com que o website fique corrompido.
→ http://fingerprint.pet-portal.eu/?menu=6
Navegador Tor
Tecnologia de anonimização dos caminhos de conexão no TCP/IP.
A principal função é falsificar endereços IP, mas como parte da anonimização do usuário, também não são obtidas impressões digitais de plug-ins, fontes, etc. do navegador da web.
→ https://www.torproject.org/
Chameleon
Esta é uma extensão para o Chrome que detecta e toma medidas contra o fingerprint. Baseia-se no conceito de fazer com que o navegador do usuário se pareça com o Navegador Tor.
→ https://github.com/ghostwords/chameleon
NoScript
Esta é uma extensão para Firefox que pode impedir a execução de JavaScript, Flash, XSS, etc. e
também é eficaz como uma contra-medida contra o rastreamento.
Se houver um script executável ao carregar uma página web, ele pode bloquear o carregamento e a execução do script.
Ao impedir a execução do JavaScript e Flash, é possível impedir que os sites coletem a maioria dos dados de fingerprint.
→ https://noscript.net/

Não é eficaz:
DNT (Do Not Track)

Quando o DNT é ativado, “dnt: 1” é adicionado ao cabeçalho de solicitação HTTP para informar o site sobre a intenção de recusar o rastreamento. Entretanto, o DNT é apenas um pedido e pode não ser executado.
De acordo com FPDetective, pesquisas mostraram que a DNT não teve efeito sobre o fingerprint.
Navegação privada
Esta é uma função de navegador da web que permite a navegação de sites sem deixar histórico.
Pode impedir a coleta de algumas informações de fingerprint, tais como fontes, mas a maior parte
continua sendo detectada.

Dica: Use o Brave ou o Firefox como navegador padrão
Mesmo sem nenhuma extensão instalada, navegadores como o Brave Browser e o Firefox confundem
algumas das informações de fingerprint por meio da randomização de dados, oferecendo maior
privacidade comparado com o Chrome ou o Microsoft Edge, por exemplo.
